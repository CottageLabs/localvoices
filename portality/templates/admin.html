Types

View

To view a Singer, Version or Song, first find their database ID using a search. Select the record type and enter the ID. Click Retrieve

Create

To add a Singer, Version or Song, first search for and click on the map to select a location, then select type of record to add. Click Create.

Update

To update a Singer, Version or Song, first find their database ID using a search. Select the record type and enter the ID. Click update. A form will appear showing the song/version or singers current listing. Edit as required and click update.

Delete

To delete a Singer, Version or Song, first find their database ID using a search. Select the record type and enter the ID. Click delete.

Link

To view and edit a Singer, Version or Songs links and relations. You will require the database ID for each item. Find the database ID using a search. Select the record type and enter the ID. Click Link. A form will appear showing the song/version or singers current links. Edit as required and click Link.
